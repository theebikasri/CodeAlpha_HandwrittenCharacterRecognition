# Handwritten Digit Recognition System

## Project Overview

This project is a Handwritten Digit Recognition System developed as part of the CodeAlpha Internship Program.

The application uses a Convolutional Neural Network (CNN) trained on the MNIST dataset to recognize handwritten digits from 0 to 9.

## Features

* Handwritten digit recognition using CNN
* Draw digits directly on a canvas
* Upload handwritten digit images
* Real-time prediction
* Confidence score display
* Streamlit web interface

## Technologies Used

* Python
* TensorFlow / Keras
* Streamlit
* NumPy
* Pillow
* Streamlit Drawable Canvas

## Dataset

MNIST Dataset

## Model Performance

* Accuracy: 98.88%
* Model: Convolutional Neural Network (CNN)
